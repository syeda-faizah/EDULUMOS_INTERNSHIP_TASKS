## Task 04: Titanic Dataset – Exploratory Data Analysis (EDA)

This repository contains the work completed for Task 04 – Exploratory Data Analysis of the Titanic dataset as part of the Data Analytics coursework.

# Files Included
File Name	Description
train.csv	Dataset used for analysis
Task_04_Titanic_EDA.ipynb	Jupyter Notebook containing all EDA code & visualizations
Task_04_Report.pdf	Final report with insights, graphs, and conclusions

# Summary of Analysis

This task performs exploratory data analysis on the Titanic dataset and focuses on identifying survival patterns based on:
Gender
Age
Passenger Class
Socio-economic factors
Visualizations include:
Survival counts
Survival by gender
Survival by class
Age distribution
Correlation Heatmap
Key outcomes highlight how demographic and socio-economic variables influenced passenger survival.
