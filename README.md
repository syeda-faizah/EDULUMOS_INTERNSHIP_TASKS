# Task 04: Titanic EDA

This folder contains the files for Task 04 - Exploratory Data Analysis of the Titanic dataset.

## Files
- `train.csv` → Dataset used for analysis
- `Task_04_Titanic_EDA.ipynb` → Jupyter Notebook with EDA code
- `Task_04_Report.pdf` → Final PDF report with graphs and insights

## Summary
EDA includes:
- Data Cleaning & Handling Missing Values
- Univariate & Bivariate Analysis
- Age Group Segmentation
- Correlation Heatmap
- Insights and Conclusion
